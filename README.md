# VOC2YOLO Converter

Простой и эффективный скрипт на Python для конвертации аннотаций из формата **PASCAL VOC** (`.xml`) в формат **YOLO** (`.txt`).

Скрипт поддерживает пакетную обработку данных, организованных по папкам, и автоматически создает файл с именами классов (`classes.txt`).

## Особенности

- 🔄 Конвертация координат bounding box из абсолютных (pixels) в нормализованные (0.0 - 1.0).
- 📂 Пакетная обработка подпапок.
- 📝 Автоматическая генерация файла `classes.txt`.
- 🐍 Написан на чистом Python (без тяжелых зависимостей, кроме стандартной библиотеки).

## Требования

- Python 3.6+
- Стандартные библиотеки: `os`, `xml.etree.ElementTree`

## Структура входных данных

Сбор **PASCAL VOC** (`.xml`) аннотаций из `input`
Результат в формате **YOLO** (`.txt`) в `output`

**! Перед началом работы необходимы папки input / output**

_⚠️ Внимание: Скрипт разработан для конвертации аннотаций с одним классом, всем объектам присваивается class_id = 0. Если у вас несколько классов, необходимо доработать функцию convert_voc_to_yolo для маппинга имен классов в ID._
